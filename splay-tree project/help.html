Допомога

Для того, щоб скористатся візуалізацією алгоритму необхідно перейти за кнопкою "Візуалізація алгоритму".

Управління

На сторінці ми можемо побачити наступні кнопки:
  • Вставити потрібно використовувати для вставки нового елемента в splay дерево, воно знаходиться одразу після текстового поля, в який можна ввести власне необхідний елемент
  • Видалити потрібно використовувати для видалення елемента з splay дерева, воно знаходиться одразу після текстового поля, в який можна ввести власне необхідний елемент
  • Знайти потрібно використовувати для знаходження певного елемента в splay дереві, воно знаходиться одразу після текстового поля, в який можна ввести власне необхідний елемент
  • Кнопка масив виконує функцію проходження splay деревом і може використовуватись в будь-який момент для того, щоб побачити масив створеного дерева
  • Кнопка "Про алгоритм" допоможе перейти до головної сторінки сайту з інформацією про алгоритм
  • Кнопка "Допомога" допоможе перейти на сторінку, на якій Ви зараз знаходитесь, що розповідає про інтерфейс
Нижче можна побачити наступні елементи управління:
  • Кнопки "Назад", "Назад на крок", "Пауза", "Вперед", "Пропустити" необхідні для управління анімацією. Кнопка "Пауза" використовується для зупинки анімації, а кнопка "Пропустити" для пропуску анімації кроку.
  • Скрол-бар з приписом "Швидкість анімації" дозволяє керувати швидкістю власне анімації.
  • Текстові поля з приписами "Ширина" та "Висота" налаштовують розмір відображення анімації. Для зміни необхідно натиснути кнопку "Змінити". Для коректного відображення анімації рекоментується встановлювати ширину з висотою з пропорціями відповідно 2/1 (наприклад, 1000 та 500, 600 та 300 і тп).
  • Для зручності користування можна також перенести нижні елементи управління наверх, використовуючи кнопку "Змінити положення кнопок".
Про доступність або недоступність для використання кнопком в моменті може свідчити їх колір. Якщо кнопка сіра - наразі її використання недоречне.
